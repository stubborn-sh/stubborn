= Data Cleanup

The broker provides configurable data retention to prevent unbounded database growth.

== Endpoints

`POST /api/v1/cleanup`::
Run cleanup across all applications. Accepts `keepVersions` parameter (default: 10)
specifying how many recent versions to retain per application.

`POST /api/v1/cleanup/applications/{name}`::
Run cleanup for a specific application.

== What Gets Cleaned

Old contract versions, their associated verifications, and deployment records are removed.
The most recent N versions (by publish date) are always retained.

== Security

Cleanup endpoints require the ADMIN role.
