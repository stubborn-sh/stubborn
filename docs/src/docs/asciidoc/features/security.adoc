= Security

Role-based access control protecting all API endpoints.

== Roles

* **READER** — Read-only access to all GET endpoints
* **PUBLISHER** — Can publish contracts, record verifications and deployments
* **ADMIN** — Full access including application registration and deletion

== Authentication

HTTP Basic authentication. CSRF disabled for stateless API usage.

== Public Endpoints

* `/actuator/health` — Available without authentication

See specification: https://github.com/stubborn-sh/stubborn/blob/main/docs/specs/006-security.md[docs/specs/006-security.md]
