# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.0.x   | Yes       |

Only the latest patch release of the current minor version receives security fixes.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Send a report to **security@stubborn.sh** with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

**Response timeline:**
- Acknowledgment within 48 hours
- Status update within 7 days
- Fix and coordinated disclosure after the patch is ready

We ask that you do not publicly disclose the vulnerability until a fix has been released. We will credit reporters in the release notes unless you prefer to remain anonymous.
